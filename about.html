About ReelTalk

An Internet Services project about the movies and TV shows we love.

The Project

ReelTalk was created as part of my Internet Services project to demonstrate how HTML, CSS, and JavaScript can be used to build a functional, interactive website.

I have always been passionate about movies and TV shows, so I decided to combine that interest with my coursework. The website features reviews of some of the greatest films and series ever made, along with interactive elements such as video players, modals, image galleries, and form validation.

Project Goals

This project was created to meet the requirements of the Internet Services course:

  • Create a multi-page website with at least 5 interlinked pages
  • Include header, logo, navigation, main body, and footer
  • Use external CSS for consistent styling
  • Add interactive elements: videos, images, tables, forms, buttons, animations, modals
  • Implement JavaScript for form validation and pop-ups
