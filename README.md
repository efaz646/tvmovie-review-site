# ReelTalk

A movie and TV review website built with HTML, CSS, and JavaScript for the Internet Services course.

## Live Demo

[View Live Site](https://YOUR-USERNAME.github.io/movie-review-site)

## Features

- 5 interlinked pages with consistent navigation
- Movie and TV show reviews with ratings
- YouTube trailer embeds
- Interactive modals for full reviews
- Image gallery with navigation
- Sign up form with validation
- Random movie picker
- Dark theme with red accent

## Pages

| Page | Description |
|------|-------------|
| [Home](index.html) | Movie cards, trailers, ratings table |
| [About](about.html) | Project description, creator info |
| [Services](services.html) | Review services, image gallery |
| [Reviews](reviews.html) | TV show reviews, trailers |
| [Sign Up](signup.html) | Registration form |

## Technologies Used

- **HTML5** - Page structure and content
- **CSS3** - Styling, animations, responsive layout
- **JavaScript** - Interactivity (modals, validation, gallery)

## Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR-USERNAME/movie-review-site.git
```

2. Open `index.html` in your browser
